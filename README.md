# 🏠 House Photo Matching Project

## 📌 Description

This project demonstrates photo matching using AutoCAD and SketchUp by recreating a real residential building from a reference image.

## 🛠 Tools Used

* AutoCAD (2D planning)
* SketchUp (3D modeling + photo match)

## 🔄 Workflow

1. Created 2D plan in AutoCAD
2. Imported into SketchUp
3. Built 3D model using Push/Pull
4. Matched perspective with real image
5. Rendered final views

## 📸 Output

* Original Image
* 3D Model
* Comparison View

## 🎯 Skills Demonstrated

* Photo Matching
* 3D Visualization
* Architectural Modeling
* Perspective Alignment
